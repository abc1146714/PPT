mybatis常用标签

用于定义sql语句：select、insert、delete、update
用于控制动态拼接：if
用于格式化输出：where、set、trim
用于定义常量：sql、include

定义sql语句

select、insert、delete、update
select标签:用作查询

控制动态拼接

if
if标签通常用于判断参数值

格式化输出

where、set、trim
where标签：有返回值时才会插入where，用于剔除and、or这种连接词
set标签：没有使用if标签时，有空值会出错；配合使用时可以剔除逗号
trim标签：prefix：前缀覆盖并增加其内容；suffix：后缀覆盖并增加其内
prefixOverrides：前缀判断的条件；suffixOverrides：后缀判断的条件

定义常量

sql、include
sql标签：查询字段或者查询条件相同时，可以将其定义为常量
include标签：用于引用定义的常量
